# beecrowd-solutions

> A repository to store my solutions for exercises from the [beecrowd](https://beecrowd.com.br) online judge.

## About the Project

This repository serves as a personal archive of my solutions to various programming challenges from beecrowd (formerly URI Online Judge). The solutions are organized into categories based on the problem type.

## Usage

To find a solution, navigate to the appropriate category folder and look for the file corresponding to the exercise ID from the beecrowd platform.

```bash
# Example structure
# /beginner/1001.py
# /strings/1234.c
```

Feel free to review the code for learning purposes, but please be mindful of the license limitations.
